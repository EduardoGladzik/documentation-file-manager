# LabDoc Automator

**Description**
This tool automates the organization of laboratory documentation files.

**What it does:**
1.  **Finds the Client:** Searches for the client folder in the "Avulsos" or "Contratados" directories on the network drive.
2.  **Creates Structure:** Generates a standard folder hierarchy for the specific RAT (Technical Attendance Report).
3.  **Moves Photos:** Automatically moves image files from the Desktop to the new RAT folder.
4.  **Generates Checklist:** Copies and renames the standard Excel checklist (`CheckListLabV7.8.xlsx`) to the new project folder.
